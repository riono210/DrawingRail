# DrawingRail

## Team
ARmStrong

## エレベーターピッチ

「プラレール等の電車模型で遊ぶ場所がない，既存のアプリは操作性が悪い」という課題を解決したい
「小学生のお子さんとその親」向けの
「場所を取らず，レールと乗り物のつくる」ことを実現する
「お絵かきレール」です。
これは「子供でも扱える操作性」があり、
「既存のアプリやプラレール」とは違って、
「自由に線路や乗り物を描き，走らせる」ことができます。
